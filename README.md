# Trels

Trels is a cross-platform (Windows & Linux) DNS record management platform featuring a user-friendly interface. 

## Features
- **Admin Panel**: Built with Next.js and shadcn/ui. Includes authentication for adding and managing DNS mappings.
- **Backend**: High-performance Go backend, compiled to a single binary.
- **Clients**: Flutter application and basic HTML/CSS/JS frontend options.
- **Future Ready**: Router DNS mapping feature is planned.

## Project Structure
- `/backend`: Go source code for the DNS server and API.
- `/frontend/admin`: Next.js admin panel.
- `/frontend/app`: Flutter client application.
- `/frontend/web`: Basic web interface.
- `/setup`: Linux and Windows installers and deployment scripts.
- `/roadmap`: Project tracking and to-do lists.
